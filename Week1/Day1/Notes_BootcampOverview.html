The Internet?

  • Internet: a bunch of computers connected to each other.
  • Servers: a computer that handles a lot of connections.

Full Stack Web Development

  • Full Stack: knowledge of front-end, back-end, and database.
  • >Every single website is built out of the same components as Full-Stack
  • Front-End: client and browser
  • Only understands HTML, CSS, and JavaScript
  • JavaScript: functionality of a website
  • CSS: Cascading StyleSheet
  • The Front-End sends an HTTP (HyperText Transfer Protocol) Request to the Back-End.
  • URLs: Uniform Resource Locator; used to locate the server.
  • Back-End: servers, behind-the scene functionality.
  • Programming languages comprise the back-end (e.g., Java, Python, C#, and even JavaScript)
  • Back-end receives the HTTP Request to the Front-End as an HTTP Response in the form of HTML, CSS, and JavaScript.
  • HTTP Request/Response Cycle: the process of HTTP Request and Response.
  • Database: stores the data.
  • E.g., MySQL, Mongo, etc.
