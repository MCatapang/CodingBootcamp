# Python vs. Javascript

## Adding Functions
- JavaScript
    ```js
        function addSum(num1, num2) {
            return num1 + num2;
        }
    ```
- Python
    ```py
        def add_sum(num1, num2):
            print(num1 + num2)
    ```

## Data Types
- Python vs. Javascript
    - Number / Int
    - String / String
    - Boolean / Boolean
    - Dictionary / Object
    - List / Arrays

## Double-Equals and Triple-Equals
- JavaScript
    ```js
    '20' == 20 // true
    '20' === 20 // false
    ```
- Python
    ```py
    # double-equals (==) doesnt exist in python
    # only triple-equals (===)
    '20' === 20 # false
    ```

## Concatenating
- Python
    ```py
    my_name = "Michael"
    my_age = 23
    phrase = f"Hi my name is {my_name} and I am {my_age}."
    ```
- JavaScript
    ```js
    '20' == 20 // true
    '20' === 20 // false
    ```
